Programming Concepts and Artefacts

03/05/2017 - Uni - Dylan Araps

Written for my programming class.

What is programming?

The act of writing a computer program.

Concepts

Sequence

Sequence is the concept of a program going from start to finish in order, the program is executed in sequence. The sequence can contain any number of actions but the actions can't be skipped. They must be executed in the order they appear.

Selection

Selection is the concept of taking a course of action based on a question/answer scenario in the code. This structure is most commonly seen in the if-else if- else structure.

Repetition

Repetition is the concept of executing the same code any number of times. Repetition is seen in loops, executing the same code until a condition is true. Repetition is also seen in functions and procedures and their reuse throughout a program.

Modularisation

Modularisation is the concept of splitting the code into reuse-able blocks, also knows as functions and procedures. Making the code modular allows you to reuse common code instead of having it repeated all over the place. The resulting code is maintainable and smaller when Modularised.

Artefacts

PROGRAM

A program is a collection of code that fulfills a purpose. For example a web browser's purpose is to load and view web pages.

FUNCTION

A section of code that takes input, manipulates it and then returns a result.

PROCEDURE

A section of code that executes and doesn't return a result.

COMMENT

Data in the program that isn't executed and is meant for documentation purposes. Like a note explaining what a section of code does.

VARIABLE

A variable is a value that can be changed.

CONSTANT

A constant is a value that can't be changed.

TYPE

A classification of data that tells the compiler how the information is intended to be used. Examples of data types are: Integer, Boolean, String and Float.

ARRAY

An array is a list of values that can be changed. Items in the array can be referenced by a numbered index, usually starting at '0'.

IF Statement

A conditional statement that executes code if a condition is true.

ELSE Statement

Execute code if the IF condition is false.

SWITCH Statement

A selection statement that executes code if a value matches another value inside the case.

WHILE/REPEAT Loop

A loop statement that repeatedly executes the code inside the statement until a condition is true.

FOR Loop

A loop statement that repeatedly executes the code inside the loop for a known number of times.
