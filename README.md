# [https://dylanaraps.com/](https://dylanaraps.com/)

My personal website.


### Setup

- Install `nodejs` and `npm`
- Install `gulp` globally.

```sh
sudo npm install -g gulp
```

- Install `gulp` and all of the<br \>
dependencies locally.

```sh
sudo npm install --save-dev

```
- Run `gulp`.
